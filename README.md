# Happy Under Cooking

React Three Fiber / Vite の小さな3D Webゲーム。

## ローカルで遊ぶ

```bash
npm install
npm run dev
```

出てきた `http://127.0.0.1:5173/` などのURLを開く。

## GitHub Pagesで公開する

1. このフォルダをGitHubのリポジトリにアップする
2. GitHubのリポジトリ画面で `Settings` を開く
3. `Pages` を開く
4. `Build and deployment` の `Source` を `GitHub Actions` にする
5. `main` ブランチにpushする

push後、`Actions` が成功するとGitHub PagesのURLで遊べる。
